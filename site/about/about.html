R6Maps.com About

Purpose

R6Maps.com is designed to help players learn Rainbow Six Siege maps.  It can be used as a reference tool on a second device while playing, or even in the UPlay or Steam overlay browsers on PCs.

Some images are taken from Ubisoft's Whiteboard Tool (others in-game), and then further edited.  Ubisoft's tool is very feature rich and allows for collaborative tactics planning.  Check it out!  This site is intended to be a light weight alternative for quick reference.

Site Help

Keyboard Shortcuts

You can switch between floors using keyboard shortcuts.  For example:

  • 0 or tilde Basement
  • 1 1st Floor
  • 2 2nd Floor
  • 3 Roof

You can toggle the following:

  • t Toggle Labels
